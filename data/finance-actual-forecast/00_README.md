事業計画とは別に、**今年（4期=2026）を主軸＋2025比較**で、法人と個人の収支を統合して整理する。

## ファイル
- `categorize.py` … 取り込みエンジン。カード追加は `SOURCES` に1行、形式違いは loader を分岐。
- `ledger_master.csv` … 全取引の統合台帳（source/owner/利用日/支払日/店名/金額/科目/セグメント）。
- `review_needed.csv` … 要確認＋個人候補を**店名単位で重複排除**、金額降順。本人が法人/個人を判定する作業用。
- `summary.py` … セグメント別・科目別・月次の集計を標準出力。
- `build_review.py` → `hantei_sheet.xlsx` … **本人判定用シート**。A=個別判定(銀行要確認＋カード大口5万↑/54行)、B=カード小口のパターン判定(科目グループ単位)、C=前提と論点。黄色セルにドロップダウンで判定を入れる。
- `_src/` … Downloads等にしか無い一時ソースのコピー（楽天銀行 全期間版など）。
- `build_pl_xlsx.py` → **`収支実績_法人個人_2025-2026.xlsx`** … 成果物。5シート（法人2025/法人2026/個人2025/個人2026/統合サマリ）。再生成は `python build_pl_xlsx.py`。
  - 各月次シートに**親科目＋子内訳行**（親＝太字合計／子＝インデント）を表示。
  - 売上＝**取引先別**。販管費・個人支出を**【固定費】【変動費】に分割**。
  - 子内訳：ソフト＝サービス別(Workspace/Claude/Zoom/freee/Cloud)、通信＝**携帯①(ドコモ約5千)/携帯②(ドコモ約8千)**/サーバー/03plus、地代家賃＝コワーキング2/レゾナンス、旅費＝宿泊/航空/交通系IC/レンタカー/タクシー等、外注費＝人名別、会議費＝**スターバックス/その他/自販機**、接待交際費＝飲食/楽天ペイ、個人カード返済＝**オリコ契約A(リボ月7,780)/契約B(分割)/アプラス/Paidy**、**Googleスマホ分割(月3,012)**。
  - 携帯は月2回線(ドコモ)を金額で①約4,953固定/②約8千変動に分割。会議費スタバ分離は categorize.py のスターバックスルール。
  - ITAは**個人支出の外貨積立**（2026/7〜200USD=月25,000想定を個人2026に計上）。
  - **BizConfort**（本人申告：法人の会議費）を2026/7〜計上：月22,000、7月のみ27,000(+5,000)。2026下期計137,000。法人_月次2026の会議費に子行で表示（統合サマリは実績1〜6月のため非表示）。
  - 「固定費一覧」シートは廃止（内訳は各月次シートに統合）。利用日ベース。Amazon・少額要確認は未反映。
  - **統合サマリ＝法人＋個人を1表に統合**。列＝年×月（2025年1〜12月＋年計／2026年1〜6月＋計）、ヘッダ2段。親科目→子内訳付き。末尾に【統合】純収支（法人営業利益＋個人収支＝役員報酬相殺後の世帯ベース）と収支外。
  - 統合純収支：2025 ▲1,212,375／2026上期 ▲1,023,932（主因は個人の外貨積立=貯蓄125万・未分類生活費。積立は資産形成のため純赤字とは別に評価）。
- `pl_summary.py` … 暫定収支を標準出力で確認する用。
