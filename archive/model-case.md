## 0. なぜ「架空」でいいか／いつ捨てるか

1社目ができるまで実事例はゼロ（HANDOFF）。それまでは**架空モデルケースで費用対効果を語る**。本人が実案件で1つでも実額（刷新費・出戻り額・取りこぼし額）を取れたら、その数字に差し替える（螺旋：1社目→ネタ→加速）。**だから数字は今は仮。桁と筋が通っていればいい。**

# A部：介護版「特養C」

## 1. ケース設定（架空）

**特別養護老人ホームC**
- 定員100名・従業員60名・情シス不在。
- 事務の**佐藤さん**に国保連請求・LIFE/加算記録・シフト・ベンダー対応が集中（属人化）。
- いま**介護ソフトの刷新**を検討中（トリガー＝システム刷新／DX焦り）。
- 経営者＝田中理事長（現場非出身・コストでしか動かない＝persona）。

> この1ケースから「出戻りの話（診断の価値）」と「年間損失の話（伴走の価値）」の2つを引く。

---

## 2. 数字①：システム刷新の出戻り（＝**診断**の費用対効果・クロージングの核）

| 項目 | 金額 | 期間 |
|---|---|---|
| 介護ソフト刷新（**うまくいった場合**） | 初期 **300万** | **6ヶ月** |
| 要件・業務構造が曖昧なまま導入 → 現場が使わない → 再設定・追加カスタマイズ・別ツール併用・再研修 | | |
| **出戻り後の現実** | **600万（2倍）** | **12ヶ月** |
| 最悪（別ソフトへ乗り換え） | 900万（3倍） | — |
| **診断 Phase0** | **30万** | 2〜4週 |

**結論（言い切り）：**
> **「300万のプロジェクトが、診断なしだと600万＝2倍になる。診断30万で、その+300万の出戻りを防げる。」**
> 診断費に対して **回避額300万 ＝ 約10倍**。

- これは経営者にも担当者にも効く（経営者＝将来コスト／担当者＝出戻りで評価を落とさない）。
- ◯倍の出どころ＝IT刷新の出戻りは珍しくない、という一般論を介護ソフトの相場（初期300万規模）に当てた保守値。**2倍を見出しに、3倍は最悪ケースとして添える。**

---

## 3. 数字②：放置している年間損失（＝**伴走**の経営メリット定量シート）

§2の「出戻り（一時）」とは別に、**毎年こぼれている損失**。梃子4本（補助金は使わない）。

| 梃子 | 現状損失/年（保守） | 改善率 r | メリット/年 | 算出の中身 |
|---|---|---|---|---|
| ①業務時間（人件費） | **70万** | 0.4 | 28万 | 二重入力・手集計 月40h × 時給1,400円 × 12 |
| ②離職コスト | **60万** | 0.3 | 18万 | 属人化事務の退職：採用・教育50万＋請求停止損の期待値 |
| ③取りこぼし加算（売上） | **180万** | 0.5 | 90万 | 記録不備・算定漏れ 月15万 ×12（介護で最大の梃子） |
| ④監査・返戻リスク | **20万** | 0.3 | 6万 | 想定返還100万 × 発生確率20% |
| **合計 B** | 330万 | | **約140万/年** | |

- **r（改善率）は楽観に振らない**。「保守的に見ても年140万」が言える方が経営者は動く。
- ③が最大の梃子＝介護は「やらないと加算＝売上が取れない」（persona）。実案件では③の実額をまず取りに行く。

## 4. 数字③：投資対効果の総合（診断＋伴走）

| | 金額 |
|---|---|
| 投資 I＝診断30万 ＋ 濃い伴走（竹75万 × 6ヶ月）480万 | **510万** |
| リターン＝出戻り回避 **300万（一時）** ＋ 運用改善 **140万/年** | |
| **回収** | 出戻り回避でまず300万、残210万を年140万で **約1.5年で回収** |
| 以降 | 年140万が利益として残り続ける（→ ライト契約で継続：L-2） |

---

## 5. ★媚びない注記（ここを誤魔化さない）

- **伴走480万は、運用改善（年140万）だけでは「1.5年回収」＝速くはない。** 単体の効率化として売ると割高に見える。
- だから伴走の費用対効果は **「刷新を正しく完遂させ、300万の出戻りを防ぐ」こととセット**で初めて立つ。売り文句は「業務がちょっと楽になる」ではなく **「刷新の失敗を防ぎ、構造ごと直して、その後も毎年140万を取り戻し続ける」**。
- 逆に **診断30万単体は文句なしに強い**（出戻り300万回避＝10倍）。**入口は診断、これがクロージングの主砲。** 伴走はその先（階段を壊さない）。
- 数字はすべて架空・保守。本人が実額を1つ取ったら差し替える。

# B部：小売版「HCチェーンH」（2026-06-24 追加・二本柱化）

> 商品は介護版と同一（診断30万・伴走竹75万×6）。違うのは**業種固有の梃子と刷新規模**。前提＝`work/persona-retail.md`（ICP＝独立系中堅・店舗10〜49／3ペルソナ A業務改革リーダー・B なんでも屋エース・C オーナー社長）。

## A1. ケース設定（架空）

**独立系ホームセンターチェーン「H」**
- 店舗**25店**・従業員約300名・独立系（大手資本傘下でない・同族 or 地場上場）。
- 情シスは**1名兼任** or 「IT得意な人」に属人化。EC/販促を兼務する**なんでも屋・中村さん（29歳）**が自動化を一人で抱える。
- いま**基幹（販売管理・在庫・受発注）の刷新**を検討中（トリガー＝基幹刷新／"第二の創業期"宣言）。
- バックオフィスが手作業：店舗報告・棚割・発注がExcelと紙、店舗間連絡は電話・FAX、経理は二重入力。

> この1ケースから「出戻りの話（診断の価値）」と「年間損失の話（伴走の価値）」を引く。介護より**規模が大きい分、刷新額も損失額も一桁上がる**＝診断の費用対効果はさらに鋭くなる。

---

## A2. 数字①：基幹刷新の出戻り（＝**診断**の費用対効果・クロージングの核）

| 項目 | 金額 | 期間 |
|---|---|---|
| 基幹（販売管理・在庫・受発注）刷新（**うまくいった場合**） | 初期 **1,500万** | **9ヶ月** |
| 要件・現場運用・店舗間の標準化が曖昧なまま導入 → 店舗ごとに使い方バラバラ・現場が紙に逆戻り → 追加開発・データ移行やり直し・再研修 | | |
| **出戻り後の現実** | **3,000万（2倍）** | **18ヶ月** |
| 最悪（プロジェクト中断・別ベンダーで仕切り直し） | 中断＋再構築で3倍 | — |
| **診断 Phase0** | **30万**（規模次第で松50万も検討 ▶） | 2〜4週 |

**結論（言い切り）：**
> **「1,500万円のプロジェクトが、診断なしだと3,000万円＝2倍になる。診断30万円で、その+1,500万円の出戻りを防げる。」**
> 出戻りの主因は**要件と業務構造の曖昧さ**（店舗ごとに運用がバラバラ・現場が使わない）。そこを**先に30万で潰す**。

- **規模が大きいほど診断が効く**＝中堅小売の刷新は介護ソフトより一桁大きく、診断の費用対効果が際立つ（30万 vs 1,500万回避）。
- ただし**1,500万級の刷新を30万診断"だけ"で de-risk しきれるとは言わない**（媚びない・§A5）。診断は「要件と構造の曖昧さ」を潰す入口。大型案件は**松診断（50万）＋伴走で実行担保**まで含めて初めて出戻りを防げる。
- A型（業務改革リーダー）には「説明可能な成果＝PoC（Phase0）」、C型（オーナー社長）には「小さく始めて効果を見てから」で同じ数字を使い分ける。

---

## A3. 数字②：放置している年間損失（＝**伴走**の経営メリット定量シート）

§A2の「出戻り（一時）」とは別に、**毎年こぼれている損失**。小売の梃子4本（介護の加算・監査の代わりに**在庫の機会損失**が最大）。

| 梃子 | 現状損失/年（保守） | 改善率 r | メリット/年 | 算出の中身 |
|---|---|---|---|---|
| ①業務時間（人件費） | **200万** | 0.4 | 80万 | 店舗報告・発注・棚割・経理の手作業/二重入力 本部+店舗 月120h × 時給1,500円 × 12 |
| ②在庫の機会損失 | **250万** | 0.5 | 125万 | 欠品による販売機会損失＋過剰在庫の廃棄・資金寝かし（**小売で最大の梃子**） |
| ③属人化・離職リスク | **80万** | 0.3 | 24万 | なんでも屋・中村さん依存：退職で自動化が止まる損＋採用教育 |
| ④EC/販促の伸ばしどころ | **100万** | 0.3 | 30万 | 雑務で攻めに回せず、伸ばせるEC売上を取りこぼし |
| **合計 B** | 630万 | | **約260万/年** | |

- ②が最大の梃子＝小売は「在庫で利益が決まる」。実案件では**まず欠品/過剰在庫の実額**を取りに行く。
- r（改善率）は楽観に振らない。25店規模なら現状損失はもっと大きい可能性が高いが、**保守側で置いて「それでも年260万」**と言える方が経営者は動く。

## A5. ★媚びない注記（小売固有）

- **小売は介護と逆で、伴走510万が"刷新1,500万の保険"として相対的に安い。** だから伴走の正当化は介護より容易（プロジェクト保険として説明しやすい）。
- ただし**「診断30万で1,500万を完全に守れる」とは言わない**。30万は要件・構造の曖昧さを潰す入口。大型刷新は松診断（50万）＋伴走で実行まで担保して初めて出戻りを防げる。**ここを盛ると報告会・現場で崩れる。**
- **25店規模なら伴走は竹（週2）でなく松（週3・110万）が妥当な局面も**。プラン選択は診断後に確定（▶ 規模別の伴走プラン早見は要検討）。
- 数字はすべて架空・保守。本人が実額（刷新費・欠品損・在庫廃棄）を1つ取ったら差し替える。
